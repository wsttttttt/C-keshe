## 使用指南
把文件BC31克隆到本地，打开BC31/Borland C++ 3.1.vbs后，直接点击run就可以运行。

## 程序说明
主要功能是实现无人机灯光秀表演的模拟
包含功能有：

1. 队形展示
2. 无人机表演（表演的队形均来自1中展示队形）
3. 实时控制（可以自己设计最终队形，提供三十个无人机操控）
4. 单体无人机路径模仿系统
5. 避障实验系统

具体细节见pdf报告
